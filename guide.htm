ISS-Conky Help and Configuration Guide

Introduction

This document describes how to install and configure "Conky-iss" , a Conky project I developed for my Ubuntu 16.04 LTS system.

After downloading and unzipping the archive, move the archive to your $HOME (~) directory and call it .conky_iss (note the preceding dot to make it a hidden folder).
Inside that folder, you will see the following files
  • Readme.md - Always read a Readme file!
  • guide.htm - This document
  • screenshot.png - Screenshot of what it looks like in my own environment
  • conky_disk.conf - Configuration for the disk usage Conky
  • conky_clock.conf - Configuration for the clock Conky
  • conky_sys.conf - Configuration for the system Conky
  • conky_functions.lua - Lua script, this contains further configuration
  • launch_conky.sh - Launcher script
  • LICENCE - The MIT licence agreement (please read before using this software but don't worry: it's not onerous!)

The scripts with the supplied configuration works well with the photograph of the International Space Station in the supplied screenshot. When using a different wallpaper, you may want to change some things to get it to look exactly how you want.

I also make use of a font called "Nasalization" if this is not what you want, then change every instance of the word "Nasalization" in every code source file to the desired font.

The ISS wallpaper and Nasalization font are both copyrighted material, although free to use I have deliberately not included them with this repo as I haven't checked the rights regarding redistribution.

Installation and Launching

After extracting the ZIP archive, move it to your $HOME directory and call it .conky_iss. That is, the full path to the folder will be $HOME/.conky_iss (or ~/.conky_iss).
Open the folder and lanuch it by double-clicking on launch_conky.sh. You can create a symbolic link to allow it to be launched from anywhere else if you wish.

Conky Configuration Parameters

Pertains to files: conky_disk.conf, conky_clock.conf and conky_sys.conf
Only the ones highlighted in red are different between the Conky set-ups in this project.
Full details for Conky configuration can be found online (that's a polite way to say STFW!).
  • alignment - Where to place the Conky window, for example top_left means top left on the left-hand monitor
  • background - Leave set to true
  • border_width - Border width surrounding the Conky window
  • cpu_avg_samples - Leave set to 2
  • default_color - Default Conky text colour, set to "gray"
  • default_outline_color - Default value is "white"
  • default_shade_color - Default value is "white"
  • draw_borders - Default value is false
  • draw_graph_borders - Default value is true
  • draw_outline - Default value is false
  • draw_shades - Default value is false
  • use_xft - Default value is true
  • font - Default font, value is "DejaVu Sans Mono:size=11"
  • gap_x - How far to the right/left to place the Conky window from the alignment above
  • gap_y - How far from the top/bottom to place the Conky window from the alignment above
  • minimum_height - Minimum Conky window height
  • minimum_width - Minimum Conky window width
  • maximum_width - Maximum Conky Window width
  • net_avg_samples - Default value is 2
  • no_buffers - Default value is true
  • out_to_console - Default value is false
  • out_to_stderr - Default value is false
  • extra_newline - Default value is false
  • own_window - Default value is true
  • own_window_class - Leave this set to "Conky"
  • own_window_type - Leave this set to "normal"
  • own_window_hints - Leave this set to "undecorated,below,sticky,skip_taskbar,skip_pager"
  • own_window_transparent - Default value is false
  • own_window_argb_visual - Default value is true
  • own_window_argb_value - Alpha, Red, Green, Blue value for window (0 = transparent)
  • double_buffer - Leave set to true unless you like flickering Conky windows!
  • stippled_borders - Default value is 0
  • update_interval - Update frequency in seconds
  • uppercase - Default value is false
  • use_spacer - Default value is "none"
  • show_graph_scale - Default value is false
  • show_graph_range - Default value is false
  • lua_load - Path and name of the Lua file (leave this alone!)

Lua functions Configuration Parameters

Pertains to file: conky_functions.lua The top part of this file contains the configuration tables for the rings and the clock. After that come the functions themselves.
A convention I have adopted is that any function whose name begins with "conky_" is one which is called from a Conky configuration file whereas a function whose names does NOT begin with "conky_" is only ever used inside this file. Any additions should hopefully follow this convention.

Conky Disk Usage Configuration

This consists of an array, one entry in the array ringdefsfor each ring to be displayed. These are the main settings for each ring to be drawn (not exhaustive) and it is recommended that any undocumented ones are left as-is:
  • parm - Shell command to run to get the value for the used amount of the ring. If blank, then use min
  • min - Minimum value, usually zero
  • max - Maximum value, must be greater than min but if not, then the Shell command in max_command will be run to get a maximum value
  • max_command - If max is less than or equal to min (see below), then run this shell command to obtain the maximum value
  • bg_colour - Background colour as an RGB value, for example, 0xff0000
  • bg_alpha - Background alpha value, for example 0.6
  • fg_colour - Foreground colour as an RGB value, for example, 0xff0000
  • fg_alpha - Foreground alpha value, for example 0.6
  • x - X-coordinate of the centre of the ring, for example 200
  • y - Y-coordinate of the centre of the ring, for example 200
  • radius - Ring radius, for example 150
  • thickness - Ring thickness, for example 24
  • start_angle - Start angle from the top of the ring going clockwise in degrees, for example 30
  • end_angle - End angle from the top of the ring going clockwise in degrees, for example 330
  • text - Display text, for example "/dev/sda5"
  • text_hoffset - Text horizontal offset, for example 144. If may be necessary to fiddle with this to get it right.
  • text_voffset - Text vertical offset, for example 120. If may be necessary to fiddle with this to get it right.
  • text_fontface - Font face for the text, for example "Nasalization"
  • text_fontsz - Font size, for example 12
  • text_colour - Text colour, for example "Black"

Conky Clock Configuration

This is a simple analogue clock with hours, minutes and seconds hands. It also gives the time in digital form as well as the current system date. The configuration is held in a Lua table called clockpars and contains the following settings:
  • parm - If this is set to "Yes" then show the clock. Otherwise, don't bother
  • clockface_colour - Colour to use for the clock face as an RGB value. Default is 0x000033
  • clockface_alpha - Alpha value for the clock face. Default is 0.8
  • x - X-coordinate in the Conky box of the centre of the clock. Default is 200
  • y - Y-coordinate in the Conky box of the centre of the clock. Default is 200
  • radius - Clock radius. Default is 160
  • border_size - Clock border; this is in addition to the radius, not included in it. Default is 4
  • border_colour - Colour of the clock border as an RGB value in hex. Default is 0xaaaaee
  • border_alpha - Alpha value for the clock border. Default is 0.8
  • hour_marker_colour - Colour to use for the twelve hour markers around the clock. Default is 0xcccccc
  • hour_marker_alpha - Alpha value for the twelve hour markers around the clock. Default is 0.8
  • hour_marker_thickness - How thick the twelve hour markers are. Default is 4
  • hour_marker_percentage - Percentage of the radius to use as the length of the hour markers. Default is 20 (percent)
  • minute_marker_colour - Colour to use for the 48 minute markers around the clock. Default is 0xaaaaaa
  • minute_marker_alpha - Alpha value for the 48 minute markers around the clock. Default is 0.8
  • minute_marker_thickness - How thick the 48 minute markers are. Default is 2
  • minute_marker_percentage - Percentage of the radius to use as the length of the minute markers. Default is 10 (percent)
  • seconds_hand_colour - Colour to use for the seconds hand. Default is 0xff0000 (red)
  • seconds_hand_alpha - Alpha value for the seconds hand. Default is 1.0
  • seconds_hand_thickness - How thick the seconds hand is. Default is 2
  • seconds_hand_percentage - Percentage of the radius to use for the length of the seconds hand. Default is 90 (percent)
  • minute_hand_colour - Colour to use for the minute hand. Default is 0x00ff00 (green)
  • minute_hand_alpha - Alpha value for the minute hand. Default is 0.6
  • minute_hand_thickness - How thick the minute hand is. Default is 3
  • minute_hand_percentage - Percentage of the radius to use as the length of the minute hand. Default is 80 (percent)
  • hour_hand_colour - Colour to use for the hour hand. Default is 0xff8800 (orange-ish)
  • hour_hand_alpha - Alpha value for the hour hand. Default is 0.6
  • hour_hand_thickness - How thick the hour hand is. Default is 4
  • hour_hand_percentage - Percentage of the radius to use as the length of the hour hand. Default is 60
  • text_voffset - vertical offset for the text (date/time). Default is 220. You may need to fiddle with this to get it right. Set to zero to not show the date/time as text.
  • text_fontface - The font to use for the date/time text display. Default is "Nasalization"
  • text_fontsz - Font size to use for date/time text display. Default it 14
  • text_colour - The colour to use when displaying the date and time. Default is "Grey"
